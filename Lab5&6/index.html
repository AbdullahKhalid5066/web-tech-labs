Part B: Float-Based Layouts

Exercise 3: Basic Float Implementation

Text wraps around floated images and creates multi-column float layouts.

Floating an image left allows text to flow around it, giving a magazine-style layout. It is essential to clear floats (see the clearfix below) to prevent parent collapse.

Floated blocks can be used to build multi-column layouts. In older designs this was a common technique before Flexbox/Grid became widely supported.
