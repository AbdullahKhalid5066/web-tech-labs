# Lab 03 & 04: CSS Fundamentals - Styling and Visual Design

This lab covers essential CSS concepts including selectors, typography, backgrounds, images, box model, gradients, shadows, transforms, and transitions.

## File Structure

- `index.html`: Main landing page with navigation to exercises
- `styles/`: CSS files organized by purpose
  - `main.css`: General styles and layout
  - `typography.css`: Font and text styling
  - `components.css`: Reusable component styles (buttons, forms, cards, etc.)
  - `effects.css`: Advanced effects (transforms, animations, shadows)
- `images/`: Image assets organized by type
  - `backgrounds/`: Background images
  - `gallery/`: Gallery images
  - `icons/`: Icon files
- `exercises/`: Individual exercise pages
  - `selectors.html`: CSS selectors practice
  - `typography.html`: Typography showcase
  - `backgrounds.html`: Background styling examples
  - `images.html`: Image styling portfolio
  - `buttons.html`: Button styling collection
  - `forms.html`: Form styling and focus states
  - `cards.html`: Card component design
  - `navigation.html`: Navigation menu styling
  - `transforms.html`: CSS transforms and transitions
  - `advanced.html`: Advanced effects (animations, gradients, shadows)
  - `portfolio.html`: Personal portfolio page styling
  - `schedule.html`: Course schedule table styling
  - `github.html`: GitHub Pages style landing page

## Learning Outcomes

By completing this lab, students will be able to:
- Understand CSS syntax, rules, and selectors
- Apply various CSS selectors and combinators
- Style text content with fonts, colors, and spacing
- Work with background properties and images
- Style images and create visual effects
- Implement CSS box model properties
- Create gradients and shadow effects
- Apply CSS transformations and transitions

## How to Use

1. Open `index.html` in your browser
2. Navigate to different exercises using the menu
3. Each exercise demonstrates specific CSS concepts
4. Modify the CSS files in `styles/` to experiment

## Technologies Used

- HTML5
